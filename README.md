# Align with Enjy — iOS Shell (Capacitor)

Native iOS wrapper for https://wellness-hub-ten.vercel.app
Built in the cloud via Codemagic (see codemagic.yaml) — no Mac required.

- Bundle ID: com.alignwithenjy.app
- Loads the live web app remotely: web deploys update the iOS app instantly.
- Native plugins: Push Notifications (APNs), Geolocation, Splash, StatusBar, Haptics.
